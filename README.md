# NoteBuddy

A personal, OneNote-inspired Angular note-taking app with:
- notebook -> section -> note hierarchy
- fast search within a section
- favorite notes
- automatic local persistence (localStorage)
- responsive 3-pane layout for desktop and mobile

## Run locally

```bash
npm install
npm start
```

Open [http://localhost:4200](http://localhost:4200).

## Production build

```bash
npm run build
```

Build output is generated at:
`dist/notebuddy-app`

## Hosting

You can host the `dist/notebuddy-app/browser` folder on static hosting providers such as Netlify, Vercel, GitHub Pages, Firebase Hosting, or AWS S3 + CloudFront.
